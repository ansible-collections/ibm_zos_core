Modules
=======

Modules can be used from the command line or in a playbook task. Ansible
executes each module on the remote target node, and collects return
values. While different modules perform different tasks, their interfaces and
responses follow similar patterns.

Module reference
----------------

Reference material contains documentation on the accepted parameters and their
expected values for each module.
